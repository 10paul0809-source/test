# 🎮 Tetris (JavaScript)

HTML5 Canvas와 JavaScript로 만든 **간단한 테트리스 게임**입니다.
별도의 라이브러리 없이 브라우저에서 바로 실행할 수 있습니다.

---

## 📌 프로젝트 소개

이 프로젝트는 테트리스의 핵심 로직인

* 블록 생성
* 이동 및 회전
* 충돌 감지
* 낙하 처리

를 직접 구현하며 **게임 로직과 Canvas 렌더링을 학습**하기 위해 제작되었습니다.

---

* **HTML5**

* **JavaScript (Vanilla JS)**
* **Canvas API**

## ▶️ 실행 방법

1. 이 저장소를 클론하거나 다운로드합니다.

   ```bash
   git clone https://github.com/your-username/tetris.git
   ```
2. `tetris.html` 파일을 브라우저에서 엽니다.
3. 바로 게임을 즐길 수 있습니다 🎉

> ❗ 서버 실행이 필요 없으며, 더블 클릭으로 실행 가능합니다.

## 🧩 주요 기능

* 7가지 테트로미노(T, O, L, J, I, S, Z)
* 블록 회전 (90도)
* 충돌 감지 (벽, 바닥, 다른 블록)
* 자동 낙하 시스템
* Canvas 기반 렌더링

## 🧠 구현 핵심 로직

* **arena(matrix)** : 쌓인 블록 상태를 저장하는 2차원 배열
* **player** : 현재 조작 중인 블록과 위치 정보
* **collide()** : 블록 충돌 감지
* **merge()** : 블록을 arena에 고정
* **rotate()** : 행렬 전치 + 뒤집기를 이용한 회전
* **update()** : 게임 루프 (낙하 + 렌더링)
